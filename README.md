# Spring Boot REST API Example

Simple REST API built with Spring Boot, Maven, and PostgreSQL.

## Technologies

- Java 17
- Spring Boot
- PostgreSQL
- Maven
- REST API

## How to run

1. Configure PostgreSQL database
2. Run `mvn spring-boot:run`
3. API available at `http://localhost:8080`
